## devicemanager edge

Manage edge computing (agents, apps, versions, configs, control)

### Examples

```
  devicemanager edge agent list
  devicemanager edge app list
  devicemanager edge version list <app-id>
  devicemanager edge config list <app-id>
  devicemanager edge control start <device-id> <app-id>
```

### Options

This command has no specific options. See subcommands.
