## devicemanager edge version deploy

Deploy an edge app version to devices

### Examples

```
  devicemanager edge version deploy <app-id> <version> --device <device-id>
  devicemanager edge version deploy <app-id> <version> --group <group-id>
```

### Options

```
  <app-id>             App ID (required positional argument)
  <version>            Version string (required positional argument)
      --device strings  Device IDs to deploy
      --group strings   Device group IDs to deploy
```
