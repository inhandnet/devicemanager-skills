## devicemanager system permission list

List device permission groups

```
devicemanager system permission list [flags]
```

### Options

```
      --cursor int    Skip N items (pagination offset)
  -h, --help          help for list
      --limit int     Number of items per page (default 20)
      --verbose int   Detail level (1-100, higher = more fields) (default 10)
```
